- Favorite music

## 👩‍🏫 Getting started

- Cloning
  - `git clone https://github.com/Hilderlan/spotcode.git`;
  - `cd spotcode`;
- Run
  - `rails db:create db:migrate db:seed`;
  - `rails s`;
